**TASK 71: Exploratory Data Analysis (EDA)**



1\. Exploratory Data Analysis (EDA) is the process of examining and understanding a dataset before performing detailed analysis.



2\. It helps identify patterns, trends, missing values, and outliers in the data.



3\. EDA improves data quality and supports better decision-making.



4\. Common EDA techniques include summary statistics, charts, and data visualization.
