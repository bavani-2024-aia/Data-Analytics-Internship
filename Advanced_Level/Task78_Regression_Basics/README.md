**TASK 78: Regression Basics**



1\. Regression is a statistical technique used to study the relationship between variables.



2\. It helps predict the value of a dependent variable based on one or more independent variables.



3\. Regression analysis is widely used in business, finance, and data science.



4\. Linear Regression is one of the most common regression techniques.



&#x20;Example: 

&#x20;    A company predicts sales based on advertising expenditure using regression analysis.
