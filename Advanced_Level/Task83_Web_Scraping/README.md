**TASK 83: Web Scraping**



1\. Web Scraping is the process of extracting data from websites automatically.



2\. It helps collect information such as product details, news articles, and market data.



3\. Web scraping is commonly performed using Python libraries like BeautifulSoup and Requests.



4\. The extracted data can be stored and analyzed for business or research purposes.



Example:



A company scrapes product prices from e-commerce websites to compare competitor pricing.
