**TASK 85: Cloud Analytics**



1\. Cloud Analytics is the process of analyzing data stored and processed on cloud platforms.



2\. It enables organizations to access large-scale computing resources without maintaining physical infrastructure.



3\. Cloud analytics supports data storage, processing, visualization, and reporting.



4\. Popular cloud analytics platforms include AWS, Microsoft Azure, and Google Cloud.



Example:

A company stores its sales data in the cloud and uses cloud analytics tools to generate business reports and dashboards.
