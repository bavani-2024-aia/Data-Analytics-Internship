TASK 10: COUNT and COUNTA



COUNT:

COUNT function is used to count only numerical values in a range.

Example: =COUNT(A1:A10)



COUNTA:

COUNTA function is used to count all non-empty cells (numbers + text).

Example: =COUNTA(A1:A10)



Difference:

&#x20;  COUNT ignores text values

&#x20;  COUNTA includes both numbers and text



These functions are useful for data analysis in Excel.
