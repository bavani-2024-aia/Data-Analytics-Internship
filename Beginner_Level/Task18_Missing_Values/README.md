**TASK 18: Missing Values Handling**



Missing values are empty or blank data in a dataset.



They can affect the accuracy of analysis, so they must be handled properly.



Common ways to handle missing values:

&#x20;1. Remove rows with missing data

&#x20;2. Fill with mean or median values

&#x20;3. Fill with default values like "Unknown"



Handling missing values helps to improve data quality.
