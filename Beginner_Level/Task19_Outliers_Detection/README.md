**TASK 19: Outliers Detection**



Outliers are data points that are very different from other values in a dataset.



They can affect analysis results, so it is important to detect them.



Examples:

&#x20; 1. A salary of 1,00,000 when most salaries are around 20,000

&#x20; 2. A mark of 1 in a class where most marks are above 50



Outliers can be detected using charts, box plots, or statistical methods.
