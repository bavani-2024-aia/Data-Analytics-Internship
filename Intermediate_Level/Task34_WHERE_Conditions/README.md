TASK 34: WHERE Conditions



&#x20;  -> The WHERE clause in SQL is used to filter records based on a specific condition.



&#x20;  -> Instead of displaying all records, it helps us retrieve only the required data.



For example, if a table contains student details, we can use a WHERE condition to display students from a particular department or with marks above a certain value.



Example:

SELECT \* FROM Students

WHERE Marks > 80;



The WHERE clause makes data retrieval more accurate and useful.
