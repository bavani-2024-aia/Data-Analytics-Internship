**TASK 36: Aggregate Functions**



&#x20;1. Aggregate functions are used to perform calculations on a group of values.

&#x20;2.  They help summarize and analyze data quickly.



Common aggregate functions include:

&#x20;  COUNT() : Counts the number of records

&#x20;  SUM() : Calculates the total value

&#x20;  AVG() : Finds the average value

&#x20;  MAX() : Returns the highest value

&#x20;  MIN() : Returns the lowest value



These functions are widely used in data analysis and reporting.



Example:



SELECT AVG(Salary) AS AverageSalary

FROM Employees;



This query calculates the average salary of all employees.
