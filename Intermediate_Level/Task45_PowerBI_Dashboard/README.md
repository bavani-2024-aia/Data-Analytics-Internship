**TASK 45: Power BI Dashboard**



1\. Power BI is a business intelligence tool used to analyze and visualize data.



2\. It helps transform raw data into interactive reports and dashboards.



3\. Power BI provides charts, tables, filters, and KPIs for better decision-making.



4\. Dashboards allow users to monitor important information in a single view.



Example: A sales dashboard can display total sales, top products, and monthly performance.
