**TASK 20: Perform Data Normalization**



1\. Data normalization is the process of organizing data to reduce redundancy.

2\. It improves database efficiency and consistency.

3\. It involves splitting large tables into smaller related tables.

4\. Normalization follows rules called normal forms (1NF, 2NF, 3NF).
