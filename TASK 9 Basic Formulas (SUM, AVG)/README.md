**TASK 9: Basic Formulas (SUM, AVG)**



In Excel, formulas are used to perform calculations on data.



Common formulas:



SUM:

Used to add values together.

Ex: =SUM(A1:A5)



AVERAGE:

Used to find the average of values.

Ex: =AVERAGE(A1:A5)



These formulas help in quick calculations during data analysis.
