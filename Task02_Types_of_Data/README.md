**TASK 2: Types of Data**



Data can be divided into different types based on its nature and usage.



1\. Structured Data:

Data that is organized in rows and columns like Excel sheets or tables.



2\. Unstructured Data:

Data that has no fixed format like images, videos, emails, or text messages.



3\. Semi-Structured Data:

Data that is partially organized like JSON or XML files.



Understanding data types helps in choosing the correct method for analysis.
