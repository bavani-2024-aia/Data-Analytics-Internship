**TASK 11: IF Condition**



IF condition is used in Excel to check whether a condition is TRUE or FALSE.



It returns one value if the condition is true and another value if it is false.



Example:

=IF(A1>50, "Pass", "Fail")



This means:

&#x20;   If value is greater than 50 → Pass

&#x20;   Otherwise → Fail



IF function is useful for decision making in data analysis.
