**TASK 14: Data Validation**



Data Validation is used in Excel to control what type of data can be entered in a cell.



It helps to avoid wrong or invalid data entry.



For example:

&#x20;   Allow only numbers between 1 to 100

&#x20;   Allow only dates in a specific range

&#x20;   Create dropdown lists for selection



It improves data accuracy and reduces mistakes while entering data.
